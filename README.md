- 문대희<br>
	죽순 팀장<br>
	웹 기반 실시간 영상 스트리밍 서비스 구현<br>
	소켓통신 기반 전체적인 라즈베리파이 서버 개발<br>

	MS Azure 감정분석, FCM 알람서비스 API 연동 및 구현<br>
